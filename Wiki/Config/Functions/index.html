FVWM Functions

Functions can be used to achieve lots of different tasks. The basic syntax for a function is:

{% highlight fvwm %} ##### # DestroyFunc FuncName # AddToFunc FuncName # + I (Action to happen Immediately) # + C (Action to happen on a mouse Click) # + D (Action to happen on a mouse Double click) # + H (Action to happen on a mouse Hold) # + M (Action to happen on a mouse Motion) ########### {% endhighlight %}
